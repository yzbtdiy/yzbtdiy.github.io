开启多线程下载

  • Chrome 或 Chromium 浏览器地址访问 chrome://flags/#enable-parallel-downloading
  • 选择 enabled 即可开启多线程下载
  • 在 Chrome 78 中，该功能仍属于实验性功能，但开启后确实能提高部分资源的下载速度
