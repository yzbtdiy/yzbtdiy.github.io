## What is yzbtdiy？

&emsp;&emsp;网名 **yzbtdiy** 诞生于 2014 年, 是由中文 “**与众不同**” 拼音的首字母和英文 “**Do it youself**” 的首字母组成, 意思是 **做与众不同的自己**, 这是一个全网唯一的用户名.

## Who am I？

&emsp;&emsp;资深宅男, Google 党, 爱好装系统, 刷机, 追番, 收集各种绿化破解软件, 喜欢极简风格.

&emsp;&emsp;微度社恐患者, 不喜欢所谓的记录生活, 不在 QQ空间, 朋友圈等社交网络发布动态, 目前常在酷安潜水, 欢迎在酷安 @yzbtdiy.

&emsp;&emsp;梦想是未来有一天能够全职投入开源软件. ( 梦想还是要有的, 万一实现了呢 )

## Why blogs？

&emsp;&emsp;这些年越来越多的网络社交走进了我的生活, 空间、朋友圈、微博等充斥着各种声音, 我一直在寻找一份属于我的宁静, 那便是一个简单的博客, 一片属于我的心灵净土.

&emsp;&emsp;自己从零开始创建一个博客是一件非常有意义的事, 自己买服务器, 买域名, 搭后台, 以至于后期的维护, 这个过程能学到很多.

&emsp;&emsp;从虚拟主机到服务器, 从 Windows 到 Linux, 从 Word 到 Markdown, 从 http 到 https, 从 WordPress 到  Typecho, 再到 Hexo、Vuepress、Hugo, 还有 Github Actions, Webhook 等,  一个小小的博客, 带给了我太多太多... ...

## Site histories：

* 2015 年开始使用 freenom 的免费域名（.tk .ml .cf .ga）和当时泛滥的 freehosting 以及开源博客系统 WordPress 建立个人网站, 当时只维护了短短几个月, 主要是搬运各种绿化破解软件, 以及一些 IT 资讯.
* 2016 年底购买了 yzbtdiy.com 域名, 并且在腾讯云购买 1 元学生主机（可能是最后一批 1元 学生主机用户）, 继续使用 WordPress, 随缘写一些学习笔记.
* 2018 年域名转入阿里云, 购买了阿里云的 10 元学生主机, 开始使用 typecho, 原因是 WordPress 对 Markdown 的支持不太友好, 其次是 typecho 更加轻量级.
* 2019 年进行了各种尝试, Web 服务器从 Apache 到 Nginx, 再到 Caddy, 升级 MariaDB 10.4, php 7.3, kernel 5.12, 加上 Redis 缓存, 在阿里云 1M 带宽上博客有了比较可观的访问速度, 2019 年 11 月 百度云加速免费 CDN 支持 https 了, 添加了 CDN 后访问速度还算理想.
* 2020 年, 考虑到 SEO 问题开始尝试静态网站生成器, 先后尝试了 hexo、hugo 和 vuepress, 最终选择了 hexo, Next 主题真心不错, 但 hugo 的效率毋庸置疑, 以后有能力的话会自己做一款 hugo 主题.
* 目前使用 Hexo + Next 主题, 静态文件提交至 github 后使用 Github Actions 自动构建, 通过 webhook 触发服务器更新, 以下域名保持同步：
