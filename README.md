# YZBTDIY 的个人博客

## 博客使用 [Hexo](https://github.com/hexojs/hexo) 驱动，主题为 [Next](https://github.com/next-theme/hexo-theme-next)

## 使用 [Github Actions](https://github.com/features/actions) + [Webhook](https://docs.github.com/webhooks/) 实现自动化部署

## 主站点托管于[阿里云](https://www.aliyun.com/)，[百度云加速](https://su.baidu.com/)提供 CDN 加速访问

## 备用站点使用 [Github Pages](https://pages.github.com/), [Netlify](https://www.netlify.com/) 的静态文件服务
