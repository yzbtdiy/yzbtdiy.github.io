## inheritance（继承）

* 在某种情况下，一个类会有“子类”，子类比原本的类（称为父类）要更加具体化
* 子类会继承父类的属性和行为，并且也可包含它们自己的
* 当一个类从多个父类继承时，我们称之为“多重继承”

## encapsulates（封装）

* 封装面向对象编程隐藏了某一方法的具体运行步骤
* 封装是通过限制只有特定类的对象可以访问这一特定类的成员，而它们通常利用接口实现消息的传入传出

## polymorphism（多态）

* 由继承而产生的相关的不同的类，其对象对同一消息会做出不同的响应
* 多态性的概念可以用在运算符重载上
